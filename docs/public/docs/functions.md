## 内置功能

WordPress 是一个应用，这就意味着，WordPress 的一些功能是不用开发，直接可以使用的，比如默认的博客功能。

为了方便 CMS 网站开发，Wizhi CMS 内置了一些功能，不用开发，直接启用就可以使用了。

## 自定义内容链接

字段为`_link_url` 的链接会自动替换 WordPress 默认的文章链接。这在需要自定义文章链接的时候特别有用。

## 文章类型存档

WordPress 中，文章、分类法项目和用户都是有自定义字段支持的。而另外一个重要的数据类型，文章类型，却没有自定义字段支持，这在开发文章类型存档页的时候，非常不友好，Wizhi CMS 添加了文章类型存档设置，Wizhi CMS 内置的文章类型均支持存档设置。Wizhi CMS 之外新建的文章类型也可以非常简单的添加自定义文章类型存档设置。
